# Legado

Esta pasta preserva apenas artefatos antigos que ainda podem ajudar na migracao.

## Banco

- `banco/atualiza-banco/`: scripts SQL historicos encontrados originalmente em `docs/atualiza banco`.
- `banco/leito.sql`: script legado de criacao de tabelas relacionadas a quartos, leitos e internacao.

Esses arquivos nao devem ser tratados como migrations oficiais da nova versao. Eles servem como referencia para entender schema, views, tabelas e regras antigas ate que uma estrategia moderna de migrations seja definida.
